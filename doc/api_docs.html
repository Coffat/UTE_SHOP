Tài liệu API UTESHOP

RESTful API chính thức — Strangler Fig modules, JWT qua HttpOnly cookies, phân quyền theo role.

Base URL http://localhost:3000/api/v1

Tổng quan API

  • Base URL: http://localhost:3000 — prefix API: /api/v1
  • Health check: GET / → { "status": "ok", "message": "UTESHOP API is running 🚀" }
  • Static files: GET /uploads/{filename} — ảnh đã upload
  • Response envelope (chuẩn): { success, message, data?, errors? }
  • Danh sách phân trang mới (GET /admin/products, GET /admin/categories): { success, message, data: Item[], meta: { page, limit, total, totalPages, ... } } — meta ở top-level, không bọc trong data.
  • Xác thực: HttpOnly cookies accessToken (15 phút) + refreshToken (7 ngày). Client gọi withCredentials: true / credentials: 'include'.
  • Refresh: POST /api/v1/auth/refresh khi access token hết hạn.
  • Rate limits: API chung 200 req/phút; login 5/15 phút; forgot-password 3/15 phút.
  • Roles: CUSTOMER, ADMIN, SALES, STORE_STAFF, WAREHOUSE_STAFF
  • Mã lỗi: 401 — thiếu token / token invalid hoặc expired; 403 — đã đăng nhập nhưng không đủ quyền; 404 — không tìm thấy; 422 — lỗi validation (express-validator); 429 — rate limit
  • Namespace Admin: Ưu tiên /api/v1/admin/products, /api/v1/admin/categories, /api/v1/admin/stock, /api/v1/admin/vouchers, /api/v1/admin/blogs. Route cũ vẫn hoạt động (backward compat) nhưng đánh dấu deprecated.

Authentication

POST/api/v1/auth/register
PUBLIC

Đăng ký tài khoản mới, gửi OTP qua email.

Parameters
Tham sốKiểuYêu cầuMô tả
fullNamestringYESHọ tên
emailstringYESEmail
passwordstringYESTối thiểu 6 ký tự
phonestringNOSố điện thoại (optional)
api.post('/api/v1/auth/register', { fullName, email, password, phone });
POST/api/v1/auth/verify-otp
PUBLIC

Kích hoạt tài khoản bằng OTP 6 chữ số.

Parameters
Tham sốKiểuYêu cầuMô tả
emailstringYESEmail
otpstringYES6 chữ số
api.post('/api/v1/auth/verify-otp', { email, otp });
POST/api/v1/auth/login
PUBLIC

Đăng nhập. Server set HttpOnly cookies accessToken (15 phút) và refreshToken (7 ngày). Rate limit: 5 req/15 phút.

Parameters
Tham sốKiểuYêu cầuMô tả
emailstringYESEmail
passwordstringYESMật khẩu
api.post('/api/v1/auth/login', { email, password }, { withCredentials: true });
POST/api/v1/auth/forgot-password
PUBLIC

Gửi OTP đặt lại mật khẩu. Rate limit: 3 req/15 phút.

Parameters
Tham sốKiểuYêu cầuMô tả
emailstringYESEmail
api.post('/api/v1/auth/forgot-password', { email });
POST/api/v1/auth/reset-password
PUBLIC

Đặt lại mật khẩu sau khi nhận OTP.

Parameters
Tham sốKiểuYêu cầuMô tả
emailstringYESEmail
otpstringYES6 chữ số
newPasswordstringYESTối thiểu 6 ký tự
api.post('/api/v1/auth/reset-password', { email, otp, newPassword });
POST/api/v1/auth/refresh

Làm mới access token từ cookie refreshToken.

api.post('/api/v1/auth/refresh', {}, { withCredentials: true });
POST/api/v1/auth/logout

Đăng xuất, xóa cookies. Dùng refreshToken cookie nếu có để revoke phiên.

api.post('/api/v1/auth/logout', {}, { withCredentials: true });
