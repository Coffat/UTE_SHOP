Tài liệu API UTESHOP 🌸

Chào mừng đến với cổng tích hợp RESTful API chính thức của UTESHOP. Hệ thống được thiết kế theo mô hình Strangler Fig Pattern, hỗ trợ xác thực bằng HttpOnly Cookies + JWT và phân quyền theo cấp độ cụ thể.

Môi trường thử nghiệm http://localhost:3000

🔑 Module: Xác thực & Đăng nhập

POST /api/v1/auth/register
PUBLIC

Đăng ký tài khoản người dùng mới. Hệ thống sẽ tạo tài khoản ở trạng thái chưa kích hoạt và gửi OTP xác minh qua Email.

Body Parameters (JSON)
Tham số Kiểu dữ liệu Yêu cầu Mô tả
fullName string YES Họ tên đầy đủ của người dùng
email string YES Email duy nhất của tài khoản
password string YES Mật khẩu bảo mật (tối thiểu 6 ký tự)
phone string YES Số điện thoại liên hệ di động 
api.post('/api/v1/auth/register', {
  fullName: 'New User',
  email: 'newuser@example.com',
  password: 'password123',
  phone: '0999888777'
});
POST /api/v1/auth/verify-otp
PUBLIC

Kích hoạt tài khoản người dùng bằng mã OTP đã nhận từ hòm thư điện tử.

Body Parameters (JSON)
Tham số Kiểu dữ liệu Yêu cầu Mô tả
email string YES Email cần xác minh kích hoạt
otp string YES Mã OTP gồm 6 chữ số vừa nhận 
api.post('/api/v1/auth/verify-otp', {
  email: 'newuser@example.com',
  otp: '123456'
});
POST /api/v1/auth/login
PUBLIC

Đăng nhập tài khoản. Nếu đúng, server phản hồi Token đã gắn sẵn trong cấu hình HttpOnly Cookie `accessToken` bảo mật cao chống XSS.

Body Parameters (JSON)
Tham số Kiểu dữ liệu Yêu cầu Mô tả
email string YES Email đăng nhập tài khoản
password string YES Mật khẩu của tài khoản 
api.post('/api/v1/auth/login', {
  email: 'vuthang@example.com',
  password: 'password123'
});
POST /api/v1/auth/logout
AUTH REQUIRED

Hủy phiên hoạt động, xóa toàn bộ HttpOnly Cookies trên trình duyệt người dùng.

api.post('/api/v1/auth/logout');

🌸 Module: Quản lý Sản phẩm & Danh mục

GET /api/v1/products
PUBLIC

Lấy danh sách các sản phẩm đang được bày bán. Cho phép lọc đa dạng theo danh mục, mức giá và phân trang.

Query Parameters
Tham số Kiểu Yêu cầu Mô tả
page number NO Trang dữ liệu hiển thị (mặc định: 1)
limit number NO Số sản phẩm mỗi trang (mặc định: 12)
categoryId string NO Lọc sản phẩm theo MongoDB ID của Danh mục
search string NO Từ khóa tìm kiếm theo tên sản phẩm 
api.get('/api/v1/products', {
  params: { page: 1, limit: 10, search: 'Lavender' }
});
GET /api/v1/products/:id
PUBLIC

Lấy chi tiết một sản phẩm theo ID hoặc Slug.

Path Parameters
Tham số Kiểu Yêu cầu Mô tả
id string YES Mã ID MongoDB hoặc slug sản phẩm (vd: lavender-dream) 
api.get('/api/v1/products/lavender-dream');

👤 Module: Người dùng & Yêu thích

GET /api/v1/users/profile
AUTH REQUIRED

Lấy hồ sơ cá nhân của người dùng đã xác thực thông qua cookie phiên làm việc.

api.get('/api/v1/users/profile');
PUT /api/v1/users/profile
AUTH REQUIRED CUSTOMER, ADMIN

Cập nhật thông tin hồ sơ của tài khoản đang đăng nhập.

Body Parameters (JSON)
Tham số Kiểu Yêu cầu Mô tả
fullName string NO Họ tên cập nhật mới
phone string NO Số điện thoại mới 
api.put('/api/v1/users/profile', {
  fullName: 'Vũ Thắng Updated',
  phone: '0999888999'
});
